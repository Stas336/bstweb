Fieldpro launch

 

 

Test name: Fieldpro launch

Internal test name: FIP_LAUNCH

 

 

 

Priority: absolutely critical (Level 1).

 

 

 

Actions performed:

 

  1. Fieldpro is launched from windows taskbar (“Run” menu) by typing the “path / executable file name”
  2. Appearance of main login screen is verified
  3. Specified database (DSN) is selected
  4. Specified user name and password are entered
  5. Launch button is pressed
  6. Main Fieldpro screen appearance is verified
