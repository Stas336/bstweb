1.       Add Filed with name FNAME

2.       Add Well with name WNAME

3.       Quick Import “S:\PHX_TEST\DATA\PHX Verification Chart.trf”

4.       For each (*.inp) file some actions are perfoming:

a.       Launch Well Explorer using menu

b.      Wait for Well Explorer

c.       Select Tree Item "/All Wells (by Field)/FNAME*/WNAME/Analysis Modules"

d.      Press Button "Import Modules from File"

e.      Wait for Express Import screen

f.        Quick Import INPFILE and uncheck "Import outputs during import file"

g.       Set Active Well Explorer

h.      Close Well Explorer

i.         Launch “Well Modules Explorer” using menu

j.        Select Tree Item "/Hydraulic Fracture ? Matrix Acid Stimulation Simulation/MNAME"

k.       Double List Select “Simulation Control Screen"

l.        Wait for SimulationControlScreen

m.    Press “Set Active” from toolbar

n.      Press “Run Simulator” from dialog

o.      Wait for button “Run Simulator” will be enable

p.      Press Global Output PHX Verification Chart using toolbar

q.      Wait for FieldproChartViewer

r.        Print To PDF And Png from FieldproChartViewer. Save output with name = “PHX Verification Chart_MNAME”

s.       Close FieldproChartViewer

t.        Close SimulationControlScreen

u.      Set Active Well Modules Explorer

v.       Close Well Modules Explorer
