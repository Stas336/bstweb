General Fieldpro Login BST procedure

DOC ID: {C7B00AFB-67C2-490e-AF6E-7A1E3406BF13}

 

1.     Windows is checked for “PRO Connection: Fieldpro.exe - Application Error”. If this message box exists – data text is copied and put to log as error. OK button is pressed.

2.     FIELDPROexpress.exe application is launched from corresponding to checked version folder. Example: I version: C:\Program Files\FIELDPRO\V7I\System\FIELDPROexpress.exe

3.     If within 30 seconds "FIELDPRO Login Screen" is not detected – corresponding exception is raised.

4.     Version is taken from corresponding label on the screen.

5.     The screen is closed.

 

This procedure is implemented as small separate test and performed each time before each test.
