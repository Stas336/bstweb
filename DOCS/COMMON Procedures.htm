PrintToPDFAndPng

 

PATH – File Path

FILE – File Name without extension

ETALON – Etalon File Path and Name without extension

 

1.       If ETALON directory does not exist then it will be created

2.       If PATH does not exist then it will be created

3.       Set Active

4.       Press button “Print”

5.       Wait For “PrintDlg” to appear

6.       For text field “Name:” Select "Adobe PDF"

7.       For “Print range” select “All”

8.       Press button “OK”

9.       Wait for “SaveAs” dialog to appear

10.   Save File

a.       For “File Name:” Set Text PATH + FILE+ “.pdf”

b.      Press button “Save”

11.   Wait For “Adobe Acrobat Professional” to appear

12.   SaveFromPdfAsPNG

a.       Set Active “Adobe Acrobat Professional”

b.      Press “SaveAS” from “File” using main menu

c.       Wait For “AdobeSaveAsDlg” to appear

d.      Select extension “PNG”

e.      Press button “Save”

f.        If AdobeSaveExist exists then press button “Yes”

g.       If AdobeAcrobat exists then select “Yes for All” and press button “Yes”

13.   Close “Adobe Acrobat Professional”

14.   If ETALON does not exist then copy FILE from PATH to path from ETALON

15.   Else  It is binary we compare files(ETALON  with PATH+FILE)

16.   If files does not equal then to inform in the report
