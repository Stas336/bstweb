Requirements

It is required to have ODBC connection (DSN named OMVProdDbUK) to MySQL OMV source database.

Phase 1

1.       Launch MIGPRO F2 screen

2.       On Tool bar click to “Import From External Database”

3.       Activate  “External Data Source Import Wizard” screen

4.       Press button “Next” 2 times

5.       For “PCF File Name” group box enter:  “S:\OMVUK\DATA\PRODDB OMV UK Test.pcf”

6.       For “Database Name” from list select:  “OMVProdDbUK ; MySQL ODBC 3.51 Driver”

7.       Press “Select Database” button

8.       For “User Name” set User Name

9.       For “Password” set Password

10.   Press “Next” button

11.   On spreadsheet set check for “Use” column (first row) (by setting focus on spreadsheet, pressing clicking “<End>” button and then “<Space>”)

12.   Click “Next”

13.   Set “Settings File Name” as “S:\OMVUK\DATA\PRODDB OMV UK Test.eds”

14.   Set “Text File Name” as “S:\OMVUK\DATA\OMV UK Update from August 2008.txt”

15.   Click “Next”

16.   Click “Finish”

17.   Wait for progress bar to disappear

 

Phase 2

1.       Click menu “File” / “Express Import”

2.       Activate “Express Import Wizard”

3.       Click “Next”  2 times

4.       For “Source File Name” Press button “ press “…” and select “S:\OMVUK\DATA\daily meter volumes.trf”

5.       Click “Next”

6.       Click “Next”

7.       Click “Next”

8.       Click “Finish”

9.       Wait for progress bar to disappear
