FIP AGIP ENI reports

 

1.  Launch SCE, Go to "User Preferences", and  set "Working Date" as "22.10.2008"

2. Prepare program for launch  for  "ECLIPSE Monthly Report"

a.  Delete Output "/Global Reports/ECLIPSE Monthly Report" if it  exists

b. Import "ECLIPSE Monthly Report" from file “{Fieldpro System Directory}\Outputs\ ECLIPSE Monthly Report.trf”

c. Open Well Explorer and Selecting “/My Current Select List” in it

d. Press button Clear Select List

e. Press Button QEZ

f. Press Next,

g. check on “Update Well Filter”, Check on “Field”

h. Press Next

j. Selecting manually Fields: "BOURI","Bu Attifel","ELEPHANT","Kathib","KK","NC-125","OO","RIMAL","UU","XX","YY"

k. Press Finish

l. Select <All Objects> in Object Navigator

3. 

a. Launch “Output Explorer”.

b. Launch  Output with "/Global Reports/ECLIPSE Monthly Report“, 

c. put in its varables "Start Date","End Date","Flag Type" "01.01.2008","31.12.2008","Operations"

d. export it to txt html

e. Verify report with formats from txt

f. Close Output Explorer

 

4. launch Well Explorer and select  Well "/All wells (Configured Tree)/A100 (*)/Bu Attifel (*)/Bu Attifel Main (*)/ A1"

 

5. For each Output from list below we do next:             

a. Launch “Output Explorer”.

b. Launch  Output with “Full Output Path” , 

c. put in its “Variables“  “Variables Values”

d. export it to Output formats

e. Verify report with formats from “Verified Formats”

f. Close Output Explorer
