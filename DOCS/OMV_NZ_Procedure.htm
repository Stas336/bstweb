Phase 0

1.      Restore SQL FIP_OMV_NZ database (use “fip_omv_nz.bak” file from \\Dataserver\Latest_Clients_Databases\OMV\ folder).

Phase 1

2.       Import meters Gross daily data for November 2002:

a)      Launch MIGPRO F2 screen using corresponding button in FIP main toolbar;

b)      Use option “Import From Excel” from action zone of opened window;

c)       Select “S:\OMV_NZ\DATA\Maui Weekly Statistics 2003.pcf“ and corresponding “S:\OMV_NZ\DATA\NZ PML381012 Weekly Prod Rpt 2002-12-01.xls” files at third screen of External Data Source Import Wizard;

d)      Set on all data sets that you want to import with the “Use” checkbox next to them at the fourth screen of External Data Source Import Wizard;

e)      Press “Next” button for several times, then press “Finish” button and wait until data will be imported;

3.       Import meters daily Gross data for 2004 year by pcf in the same way as above using “S:\OMV_NZ\DATA\Maui Weekly Statistics - Yearly.pcf “ and corresponding “S:\OMV_NZ\DATA\NZ PML381012 Weekly Prod Rpt 2004-12-31.xls” files.

4.     Import meters daily Gross data for 2004 year by pcf in the same way as above using “S:\OMV_NZ\DATA\Maui Reconciled_corrected.pcf“ and corresponding “S:\OMV_NZ\DATA\ MAUI - reconciliation 2008.xls” files.

5.       Import “S:\OMV_NZ\DATA\Maui Condensate (Gross).trf” and “S:\OMV_NZ\DATA\Maui Oil (Gross).trf” and “S:\OMV_NZ\DATA\Meters pcf imported data report_v7e.trf”reports to FIELDPRO.

6.       Output and verify

Phase 2

1.       Log in as Administrator

2.       Launch Well Explorer, find All Wells(by Field)\Pohokura\POW-01\Data Components\POW-01 Wellbore Configuration Component and open it

3.       On the Equipment Groups level launch Screen Setup dialog, sort items by Name, find “Type of String” column, change column order to 3, press OK button.

4.       Set “Type of string”=Casing for the following Equipment Groups: Conductor, 16" Surface Casing, 10 3/4" Intermediate Casing.

5.       Set “Type of string”=Liner for the following Equipment Groups: 7 5/8" Liner, 5" Liner.

6.       Set “Type of string”=Production String for 5 1/2" x 5" Completion String

7.       Open 5 1/2" x 5" Completion String by double clicking on it

8.       Launch Screen Setup dialog, sort items by Name, find “Substitute Label” column, change column order to 3. Find “Tool Based Stretching” column, change column order to 4, press OK button.

9.       Enter the following “Substitute Label” data:

10.    For “TUBING HANGER NIPPLE Cameron SSMC x 5 1/2" c/w Petroline 4.750" QXT Profile” tool enter A

11.    For “HALLIBURTON SP-e 5 1/2"" TRSCSSV c/w 4.562" RPT Profile & 4.650" No-Go” tool enter B

12.    For “HALLIBURTON 4.665" FBN W/LINE L/NIPPLE c/w 4.69" Honed Seal Bore” tool enter C

13.    For “PROMORE "MORE T" PERMANENT PRESSURE AND TEMPERATURE GAUGE” tool enter D

14.    For “HALLIBURTON 'OO' ANNULAR SET TUBING ANCHOR” tool enter E

15.    For “HALLIBURTON 4.00" RPT WIRELINE LANDING NIPPLE c/w 4.090" No-Go” tool enter F

16.    For “STEM SEAL ASSEMBLY EQUIPED WITH 18 TEFLON SEAL STACKS & X-OVER.” tool enter G

17.    Enter the following “Tool Based Stretching” data:

18.    For “HALLIBURTON SP-e 5 1/2"" TRSCSSV c/w 4.562" RPT Profile & 4.650" No-Go” tool enter 5

19.    For “HALLIBURTON 4.665" FBN W/LINE L/NIPPLE c/w 4.69" Honed Seal Bore” tool enter 5

20.    For “PROMORE "MORE T" PERMANENT PRESSURE AND TEMPERATURE GAUGE” tool enter 5

21.    For “HALLIBURTON 'OO' ANNULAR SET TUBING ANCHOR” tool enter 5

22.    For “HALLIBURTON 4.00" RPT WIRELINE LANDING NIPPLE c/w 4.090" No-Go” tool enter 5

23.    For “STEM SEAL ASSEMBLY EQUIPED WITH 18 TEFLON SEAL STACKS & X-OVER.” tool enter 5

24.     Select “16" Surface Casing” Equipment Group, find “Casing Shoe” tool inside, change “Substitute Label” to “16" Shoe 606.5m” for this tool.

25.    Select “10 3/4" Intermediate Casing” Equipment Group, find “Casing Shoe” tool inside, change “Substitute Label” to “10 3/4" Shoe 2695m” for this tool.

26.    Select “7 5/8" Liner” Equipment Group, find “Casing Shoe” tool inside, change “Substitute Label” to “7 5/8" Shoe 5588m” for this tool. Find “Liner Hanger” tool, change “Substitute Label” to “7 5/8" TOL 2528m”

27.    Select “5" Liner” Equipment Group, find “Casing Shoe” tool inside, change “Substitute Label” to “5" Shoe 6455.4m” for this tool. Find “Liner Hanger” tool, change “Substitute Label” to “5" TOL 5413.3m”

28.    Save changes in the component and close it.

 

 

 

 

Phase 3

 

1.     Set Current Date "11.02.2009"

a.     Launch System Configuration Explorer using main menu

b.    Wait For System Configuration Explorer to appear

c.     Maximize System Configuration Explorer

d.    Select Tree Item “/User Preferences”

e.     For “Working Date” set value = "11.02.2009"

f.     Close System Configuration Explorer

2.     Set Active Well

a.     Launch Well Explorer using main menu

b.    Select Tree Item “/All Wells (by Field)/Pohokura (14)/POW-01/Data Components”

3.     Check Zoom For Report

a.     Run Output From Explorer

                                                           i.    Launch Output Explorer using main menu

                                                          ii.    Select Tree Item “/All Reports/Administrator (*)/Completion Report”

                                                         iii.    Press View button from toolbar

b.    Enter Properties For Report (“Revision Number” – “1”) using “Enter Properties For Report” dlg

c.     Press button View from “Enter Properties For Report” dlg

d.    Wait for Report to appear

e.     Set Zoom "25 %"

f.     Press Page Setup from toolbar

g.    Wait for “Page Output Setup” dlg to appear

h.     Select Page List "Output Setup"

i.      Select Fit To Page "Fit Whole Report to"

j.      Press Print Setup

k.     Wait For “Page Setup” dlg to appear

l.      Select Orientation "Landscape"

m.   Press OK button from “Page Setup” dlg

n.     Press OK button from “Page Output Setup” dlg

o.    Wait For Report to appear

p.    Set window size (500, 500)

q.    Verify Bitmap (File: “S:\OUTPUTS\FIP_OMV_NZ\BITMAPS\Completion_Report.PNG”, Etalon: “S:\OUTPUTS\FIP_OMV_NZ\BITMAPS\ETALON\Completion_Report.PNG”)

r.      Close and don’t save Report

4.     Output Report

a.     Repeat step 3.a

b.    Print Report to PDF (File: “S:\OUTPUTS\FIP_OMV_NZ\REPORTS\Completion Report.pdf”)  and Save it to PNG (File: “S:\OUTPUTS\FIP_OMV_NZ\REPORTS\Completion Report.png”)

c.     Verify with Etalon: “S:\OUTPUTS\FIP_OMV_NZ\REPORTS\ETALON\Completion Report.png”

d.    Close Report

e.     Close Output Explorer
