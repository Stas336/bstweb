Instruction #1 (data component creation steps)

 

1.       Activate wells explorer

2.       Select tree item “/All Wells (by Field)/FNAME (*)/WNAME/Data Components”

3.       Press button “Create”

4.       Press button comp_type

5.       For last creation item set name = CNAME

 

Instruction #2 (adding references)

Parameters:

                Path: string - indicates what path should be selected in the tree in the system configuration explorer

                Names: list of string – indicates list of object names should be added (note normal mode: Name, ID  and Abbreviation is the same)

                Button Name: string – indicates what button should be pressed to add new reference

 

1.       Open System Configuration Explorer using menu

2.       Select Path

3.       Press button “Button name”

4.       For {"Name", "ID", "Abbreviation"} columns set text = PNAME for newly created item

5.       Close System Configuration Explorer

5.       For last creation item set name = MNAME

Production Daily and Monthly Data Entry (Step 3)

 

This procedure / phase is performing Main Production Daily procedure (see below) for all of the wells from “Well – Field correspondence map”

Setup Forecast (Step 9)

 

This procedure / phase is performing Main Forecast procedure (see below) for all of the wells from “Well – Field correspondence map”

 

Main Forecast procedure

1.       Open Well Explorer using menu

2.       Add data component using Instruction #3 (Analysis Module creation steps)  (add MNAME component for FNAME field for WNAME well)

a.       If  forecast module Exponential, then MNAME = “WNAME Exponential Forecast”

b.      If  forecast module Harmonic, then MNAME = “WNAME Harmonic Forecast”

c.       If forecast module Linear, then MNAME = “WNAME Linear Forecast”

d.      If forecast module Hyperbolic, then MNAME = “WNAME Hyperbolic Forecast

3.       Double click MNAME in list

4.       Wait for Production Forecast Module Input screen to appear

5.       Press button “Channel Source Destination” from screen’s toolbar

6.       For "Source Data Flag Type" column set value = "Operations"

7.       For " Base Data Flag Type" column set value = "Operations"

8.       Press button “Next” from main toolbar

9.       Wait for Production Forecast module Analysis Setup screen to appear

10.    For “Regression Type” column for Result period set type

a.       If forecast module Exponential, then set type = Exponential

b.      If forecast module Harmonic, then set type = Harmonic

c.       If forecast module Linear, then set type = Linear

d.      If forecast module Hyperbolic, then set type = Hyperbolic

11.     Press button “Source” from toolbar

12.    Wait for Chart screen

13.    Compare current bitmap with  etalon

a.       If etalon does not exist, then add current bitmap to etalon

b.      If etalon and current bitmap does not identical, then add current bitmap to problem

14.    Close Chart screen

15.    Press button “Analysis” from toolbar

16.    Repeat step 12-14

17.    Press button “Result” from toolbar

18.    Repeat step 12-14

19.    Save Production Forecast module Analysis Setup screen

20.    Close Production Forecast module Analysis Setup screen

21.    Close Production Forecast Module Input screen
